# Arcana Executive OS — UI/UX Design Pack (Pilot 0.1)

Detta paket mappar direkt mot leverablerna i `Arcana Executive OS Design.txt`.

## Innehåll
- `sitemap.md` — navigation map + informationsarkitektur
- `wireframes.md` — låg-fidelitet skärmstrukturer per huvudsektion
- `component-library.md` — komponentkatalog med states och användning
- `design-tokens.json` — maskinläsbara tokens
- `design-tokens.css` — CSS-variabler för tokens
- `user-journeys.md` — centrala Owner/Staff-flöden
- `interactions.md` — interaktionsregler (filter, tabeller, modaler, notifieringar)

## Mål
- Snabb orientering: max två klick till kärnfunktioner
- Tydlig riskprioritering: Reviews (L3) och Incidents (L4-L5 + SLA)
- Konsekvent språk och actions mellan Overview, Templates, Risk, Audit, Team, Settings
